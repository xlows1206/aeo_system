# 任务
请你从文档中提取行政处罚相关信息：
- 时间 (time)
- 被罚款单位 (company)
- 罚款金额 (amount)，单位为元

# 判定规则
1. **通过 (pass)**: 文档明确显示该单位在规定时间内无相关行政处罚记录。
2. **不通过 (failed)**: 提取到行政处罚记录，且金额或性质不符。
3. **错误 (error)**: 图片模糊、找不到单位名称、或资料不全。

# 输出格式
请按 JSON 格式输出，**严禁**包含任何解释性文字：

成功示例 (无处罚)：
{
  "status": "pass",
  "reason": "未发现行政处罚记录"
}

失败示例 (发现处罚)：
{
  "status": "failed",
  "time": "2022-05-20",
  "company": "{单位名称}",
  "amount": "50000.00",
  "reason": "发现 2022 年有行政处罚记录"
}

错误示例 (无法识别)：
{
  "status": "error",
  "reason": "无法找到行政处罚数据或图片过于模糊"
}
