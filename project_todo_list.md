# AEO System Project Todo List

## 项目目标
完善 AEO 系统中“单项标准 (standard_id=6)”的业务匹配逻辑，确保企业根据自身业务主体性质（物流、仓储、报关、外贸）动态加载和展示检测项目，并确保系统环境与数据持久化稳定。

- **系统架构**: 数据库持久化、API 容错与缓存管理。

## 处理流程
1. 用户在公司设置勾选业务类型 -> 前端计算 bind_projects -> 同时发起请求保存。
2. 后端接收请求 -> 更新 `company_info` 表 -> `types` 和 `bind_projects` 存储为逗号字符串。
3. 用户进入资料管理 -> `allProjects?standard_id=6` 接口读取 `bind_projects` -> 执行 `WHERE IN` 过滤。
4. **规范化展示**：前端通过 `[-]` 和 `[、]` 正则精准识别分类，实现自动归组展示。
