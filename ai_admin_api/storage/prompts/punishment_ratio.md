# 任务
请你识别报关单文件中是否包含“经自查”关键字。

# 判定规则
1. **通过 (pass)**: 成功识别到“经自查”关键字。
2. **不通过 (failed)**: 未识别到相关关键字。
3. **错误 (error)**: 图片模糊、或资料不属于报关单。

# 输出格式
请按 JSON 格式输出，**严禁**包含任何解释性文字：

成功示例：
{
  "status": "pass",
  "reason": "找到关键字：经自查"
}

失败示例：
{
  "status": "failed",
  "reason": "未找到关键字：经自查"
}

错误示例：
{
  "status": "error",
  "reason": "无法辨认图片内容或资料不属于报关单"
}
